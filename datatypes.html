NumPy Data Types

A Visual Guide to Understanding and Converting dtypes

Data Types in Python

By default, Python has these fundamental data types which NumPy builds upon:

  • strings - used to represent text data, the text is given under quote marks. e.g. "ABCD"
  • integer - used to represent integer numbers. e.g. -1, -2, -3
  • float - used to represent real numbers. e.g. 1.2, 42.42
  • boolean - used to represent True or False.
  • complex - used to represent complex numbers. e.g. 1.0 + 2.0j, 1.5 + 2.5j

NumPy's Data Types (`dtype`)

NumPy has its own, more specific data types that are optimized for numerical operations. Every NumPy array is a grid of elements of the same type, known as its `dtype`.

Here are some of the most common `dtype` characters:

  • i - integer (e.g., `i4` for 32-bit, `i8` for 64-bit)
  • f - float (e.g., `f4` for 32-bit, `f8` for 64-bit)
  • b - boolean
  • U - Unicode string
  • S - byte string
  • c - complex float

Checking an Array's Data Type

You can easily check the data type of a NumPy array using the .dtype attribute.

Visualizing `arr.dtype`

1.1
2.5
3.0
4.8
dtype('float64')

Code

import numpy as np

# NumPy automatically detects the type
arr = np.array([1.1, 2.5, 3.0, 4.8])

# Check the data type
print(arr.dtype)

# Output: float64

Changing an Array's Data Type

You can create a copy of an array with a different data type using the .astype() method. This is useful for converting between floats, integers, strings, etc.

Using `arr.astype('i')`

Original Array (float)

1.1
2.5
3.9

New Array (integer)

1
2
3

Note: The decimal parts are truncated (cut off), not rounded.

Code

import numpy as np

arr_float = np.array([1.1, 2.5, 3.9])
print(f"Original dtype: {arr_float.dtype}")

# Create a new array by changing the type
arr_int = arr_float.astype('i') # or np.int32

print(f"New array: {arr_int}")
print(f"New dtype: {arr_int.dtype}")

# Output:
# Original dtype: float64
# New array: [1 2 3]
# New dtype: int32

Using the correct `dtype` can save memory and improve performance.
