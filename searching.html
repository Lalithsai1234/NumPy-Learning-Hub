NumPy Array Searching

A Visual Guide to Finding Your Data

1. Finding Indices with `np.where()`

The `np.where()` method returns the indices (locations) of elements where a given condition is true. It's like asking "Where in the array are the values that match my rule?".

Find where `arr == 4`

10
21
32
43
54
45

The condition is true at indices 3 and 5.

Code

import numpy as np

arr = np.array([1, 2, 3, 4, 5, 4])

# Find all indices where the value is 4
indices = np.where(arr == 4)

print(indices)

Output

(array([3, 5]),)

The output is a tuple containing an array of the indices.

2. Finding Insertion Points with `np.searchsorted()`

The `np.searchsorted()` method works on sorted arrays and tells you where a value should be inserted to maintain the sorted order. By default (`side='left'`), it gives the first possible index.

Find where to insert the value `7`

10
21
42
83
94

To keep the array sorted, `7` should be inserted at index `3`.

Code

import numpy as np

# searchsorted only works on sorted arrays
arr = np.array([1, 2, 4, 8, 9])

# Find the index where 7 should be inserted
index = np.searchsorted(arr, 7)

print(index)

Output

3

3. Using `side='right'` for the Highest Index

You can use the `side='right'` argument in `np.searchsorted()` to get the index on the right-hand side of any existing matching elements. This gives you the last possible position for insertion.

Find where to insert `8` using `side='right'`

40
61
82
83
94

The rightmost insertion point for `8` is at index `4`.

Code

import numpy as np

arr = np.array([4, 6, 8, 8, 9])

# Find the index on the left side (default)
left_index = np.searchsorted(arr, 8, side='left')
print(f"Left side index: {left_index}")

# Find the index on the right side
right_index = np.searchsorted(arr, 8, side='right')
print(f"Right side index: {right_index}")

Output

Left side index: 2 Right side index: 4

Use `where` for conditional searching and `searchsorted` for fast searching in sorted data.
