NumPy Array Iteration

A Visual Guide to Looping Through Your Data

1. Basic Iteration with `for` Loops

Using a standard `for` loop is the most basic way to iterate. When used on a multi-dimensional array, it will iterate over the elements of the first axis (i.e., row by row).

Looping row by row

1
2
3
4
5
6

First iteration gets `[1 2 3]`, second gets `[4 5 6]`.

Code

import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])

print("Iterating over the array:")
for row in arr:
    print(row)

Output

[1 2 3] [4 5 6]

2. Iterating with `np.nditer`

For more advanced iteration, NumPy provides `np.nditer`. This is a more efficient iterator that visits every single element of an array, regardless of its dimensions.

Visiting each element one by one

1
2
3
4
5
6

The loop visits 1, then 2, then 3, then 4, and so on.

Code

import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])

print("Iterating with nditer:")
for element in np.nditer(arr):
    print(element, end=' ') # print on same line

Output

1 2 3 4 5 6

3. Iterating with `np.ndenumerate`

Sometimes you need both the value and its index (coordinates) during iteration. `np.ndenumerate` provides this by yielding a tuple of `(index, value)` for each element.

Getting index and value together

1
2
3
4
5
6

First iteration gets `(0, 0)` and `1`.

Code

import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])

print("Iterating with ndenumerate:")
for index, value in np.ndenumerate(arr):
    print(index, value)

Output

(0, 0) 1 (0, 1) 2 (0, 2) 3 (1, 0) 4 (1, 1) 5 (1, 2) 6

Choose the right iteration method for your task to write clean and efficient code.
