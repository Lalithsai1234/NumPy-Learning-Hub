NumPy Visual Learning Hub
Welcome to the NumPy Visual Learning Hub, an interactive, single-page website designed to help beginners understand the core concepts of the NumPy library through dynamic and intuitive visualizations.

🚀 About This Project
This project provides a series of self-contained lessons on fundamental NumPy topics. Instead of just reading text, users can see how array manipulations work, making complex topics like broadcasting, indexing, and reshaping easier to grasp. The main goal is to provide a hands-on, visual-first approach to learning.

✨ Features
The learning hub is organized into 10 distinct, easy-to-follow modules:

Creating Arrays: Learn to create 1D, 2D, and 3D arrays from scratch.

Indexing & Slicing: Master how to access specific elements and extract subsections of your arrays.

Data Types: Explore NumPy's dtypes and learn how to check and convert between them.

View & Copy: Understand the critical difference between a view and a copy to avoid common bugs.

Shape & Reshape: Learn what an array's shape is and how to change its dimensions.

Iterating: Discover different and efficient ways to loop through array elements.

Join & Split: Combine and break apart arrays using functions like concatenate and split.

Searching: Find elements in an array based on conditions (where) or in sorted arrays (searchsorted).

Sorting: Learn how to order your data and understand the behavior of the sort function.

Filtering: Select the data you need by creating and applying boolean masks.

🛠️ How to Use
Clone or download this repository to your local machine.

Open the main folder.

To start the application, open the index.html file in your web browser. This file is the main hub that links to all the other lesson pages.
