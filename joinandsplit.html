Joining Arrays

Joining means putting the contents of two or more arrays into a single array. NumPy provides several functions for this, each behaving differently.

1. `np.concatenate`

The most general-purpose joining function. It joins a sequence of arrays along a specified axis.

Code

import numpy as np

a = np.array([1, 2])
b = np.array([3, 4])

# Concatenate along axis 0 (default for 1D)
result = np.concatenate((a, b))
# result is: [1 2 3 4]

2. `np.vstack` (Vertical Stack)

Stacks arrays in sequence vertically (row-wise). This is equivalent to `concatenate` along axis 0 for 2D arrays.

a = np.array([1, 2])
b = np.array([3, 4])

# Stack vertically
result = np.vstack((a, b))
# result is:
# [[1 2]
#  [3 4]]

3. `np.hstack` (Horizontal Stack)

Stacks arrays in sequence horizontally (column-wise). This is equivalent to `concatenate` along axis 1.

# Stack horizontally

# result is:
# [[1 2]
#  [3 4]]

Splitting Arrays

Splitting is the reverse of joining: breaking one array into multiple smaller ones.

1. `np.split`

Splits an array into multiple sub-arrays. You can specify the number of equal-sized sub-arrays or the exact split points.

Code

x = np.arange(1, 7)
# x is: [1 2 3 4 5 6]

# Split into 3 equal arrays
result = np.split(x, 3)
# result is: [array([1, 2]), 
#             array([3, 4]), 
#             array([5, 6])]

2. `np.hsplit` (Horizontal Split)

Splits an array horizontally. This is equivalent to `split` with `axis=1`.

Code

x = np.arange(1, 9).reshape(2, 4)
# x is:
# [[1 2 3 4]
#  [5 6 7 8]]

# Split into 4 arrays
result = np.hsplit(x, 4)
# result is a list of 4 arrays:
# [ [[1],[5]], [[2],[6]], ... ]
