Filtering NumPy Arrays

A Visual Guide to Selecting Data Based on Conditions

The Core Idea: A Boolean Mask

Filtering in NumPy works by creating a "mask" array of `True` and `False` values. You create this mask by applying a condition directly to an array.

Creating a mask for the condition `arr > 4`

Code

import numpy as np

arr = np.array([1, 8, 3, 5])

# Create a boolean mask based on a condition
mask = arr > 4

print(mask)

Output

[False True False True]

Applying the Mask to Filter Data

Once you have the mask, you can use it as an index on the original array. This will return a new array containing only the elements where the mask was `True`.

Using the mask `[F, T, F, T]` to filter

1. Original Array & Mask

1
8
3
5

2. New Filtered Array

5

Code

arr = np.array([1, 8, 3, 5])
mask = arr > 4

# Use the mask to filter the array
filtered_arr = arr[mask]

# You can also do it in one line:
# filtered_arr = arr[arr > 4]

print(filtered_arr)

Output

[8 5]

When Should You Use Filtering?

Filtering is one of the most common tasks in data analysis and preparation.

  • Data Cleaning: To remove invalid or unwanted values. For example, finding all entries greater than 0 in a dataset of prices.
  • Data Selection: To select a specific subset of your data for analysis. For example, selecting all sensor readings that are within a normal operating range.
  • Counting: To count how many elements in an array meet a certain condition. For example, `np.count_nonzero(arr > 100)` or `np.sum(arr > 100)`.

Filtering is a powerful way to select exactly the data you need.
