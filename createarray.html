Visualizing NumPy Dimensions

An Interactive Guide to Creating 1D, 2D, and 3D Arrays

1D Array (Vector)

The simplest form of a NumPy array. It's a single list of elements, like a vector in mathematics. It has one axis (axis 0).

Shape: (4,)

1
2
3
4

Code

import numpy as np

# Create a 1D array
arr_1d = np.array([1, 2, 3, 4])

print(arr_1d.ndim) # 1
print(arr_1d.shape) # (4,)

2D Array (Matrix)

A 2D array is like a table or a spreadsheet, with rows and columns. It has two axes: axis 0 (for rows) and axis 1 (for columns).

Shape: (2, 3)

1
2
3
4
5
6

Code

# Create a 2D array (2 rows, 3 columns)
arr_2d = np.array([[1, 2, 3], 
                   [4, 5, 6]])

print(arr_2d.ndim) # 2
print(arr_2d.shape) # (2, 3)

3D Array (Tensor)

A 3D array can be visualized as a collection of 2D arrays (matrices) stacked together. It has three axes: axis 0 (depth), axis 1 (rows), and axis 2 (columns).

Shape: (2, 2, 2)

Visualized as 2 matrices of shape (2, 2)

Axis 0, Index 0

Code

# Create a 3D array 
# (2 matrices, each with 2 rows and 2 columns)
arr_3d = np.array([[[1, 2],
                    [3, 4]],
                   
                   [[5, 6],
                    [7, 8]]])

print(arr_3d.ndim) # 3
print(arr_3d.shape) # (2, 2, 2)

Visualizing dimensions makes complex data structures easier to understand.
