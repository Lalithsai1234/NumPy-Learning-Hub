NumPy: Shape and Reshape

A Visual Guide to Structuring Your Arrays

Part 1: What is an Array's Shape?

The shape of an array describes its dimensions. It tells you how many elements you have along each axis. For a 2D array, this is simply `(rows, columns)`.

An array with 2 rows and 4 columns

Shape: (2, 4)

Code

import numpy as np

arr = np.array([[0, 1, 2, 3],
                [4, 5, 6, 7]])

# Check the shape
print(arr.shape)

# Output: (2, 4)

Part 2: What is Reshaping?

Reshaping means changing the shape of an array. You can change the number of rows and columns, but the total number of elements must remain the same.

The Golden Rule: The product of the dimensions must be constant. For example, an array with 12 elements can be shaped as `(1, 12)`, `(12, 1)`, `(3, 4)`, or `(4, 3)` because `1*12 = 3*4 = 12`.

import numpy as np

# A 1D array with 12 elements
arr_1d = np.arange(12)
print(f"Original shape: {arr_1d.shape}")

# Reshape it into a 2D array with 3 rows, 4 cols
arr_2d = arr_1d.reshape(3, 4)

print("Reshaped array:")
print(arr_2d)
print(f"New shape: {arr_2d.shape}")

Using -1 for an Unknown Dimension

If you provide one dimension as -1, NumPy will automatically calculate the correct size for that dimension based on the array's total number of elements.

arr.reshape(2, -1)

NumPy calculates this as (2, 6)

arr.reshape(-1, 4)

NumPy calculates this as (3, 4)

Code

arr = np.arange(12)

# Reshape to have 2 rows, and let NumPy
# figure out the number of columns.
arr_2_rows = arr.reshape(2, -1)
print(arr_2_rows.shape)
# (2, 6)

# Reshape to have 4 columns, and let NumPy
# figure out the number of rows.
arr_4_cols = arr.reshape(-1, 4)
print(arr_4_cols.shape)
# (3, 4)

Shaping and reshaping are essential for preparing data for analysis and machine learning.
