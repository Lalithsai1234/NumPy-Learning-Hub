The NumPy `sort()` Method

A Visual Guide to Ordering Array Data

How `np.sort()` Works

The `np.sort()` function takes an array and returns a new, sorted copy of it. This is a crucial point: the original array is not modified in any way.

Code

import numpy as np

original_arr = np.array([9, 2, 6, 1])
print(f"Original array: {original_arr}")

# np.sort() returns a new, sorted copy
sorted_arr = np.sort(original_arr)

print(f"Sorted copy:    {sorted_arr}")
print(f"Original array is NOT changed: {original_arr}")

Output

Original array: [9 2 6 1] Sorted copy: [1 2 6 9] Original array is NOT changed: [9 2 6 1]

When Should You Use `np.sort()`?

You should use this method whenever you need an ordered version of your data but you also need to preserve the original order for other purposes.

  • Data Analysis: To find the smallest or largest values (min/max), percentiles, or the median of your data without losing the original sequence.
  • Preparing for Searching: Functions like `np.searchsorted()` require the array to be sorted. You would first create a sorted copy to use with that function.
  • Safe Operations: When you are writing a function and want to avoid side effects. By creating a sorted copy, you ensure your function doesn't accidentally change the original data that was passed into it.

Use `np.sort()` for a safe, non-destructive way to get an ordered version of your array.
