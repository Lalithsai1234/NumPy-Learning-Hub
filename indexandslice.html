Code

import numpy as np
arr = np.arange(12).reshape(3, 4)

# Step 1: Basic indexing (row 1, col 2)
element = arr[1, 2]
# element is 6

# Step 2: Slicing a region
# Rows 0-1, Cols 1-2
sub_array = arr[0:2, 1:3]
# [[1 2]
#  [5 6]]

# Step 3: Get a full row or column
# The ':' means "get everything"
row = arr[1, :]   # Get row 1 -> [4 5 6 7]
col = arr[:, 2]   # Get col 2 -> [2 6 10]

# Step 4: Slicing with a step
# Format is start:stop:step
# Get every 2nd row and every 2nd column
stepped = arr[::2, ::2]
# [[0  2]
#  [8 10]]

2. Integer Array Indexing

Also known as "fancy indexing," this allows you to construct a new array by picking elements using lists or arrays of indices.

Accessing `arr[[0, 2, 3]]`

Code

arr = np.array([10, 20, 30, 40, 50])

# Provide a list of indices to select
indices = [0, 2, 3]
result = arr[indices]

# result is: [10 30 40]

# You can also use it for 2D arrays
arr2d = np.arange(9).reshape(3, 3)
# Select rows 0 and 2
rows = arr2d[[0, 2]]
# rows is:
# [[0 1 2]
#  [6 7 8]]

Code

arr = np.arange(1, 7).reshape(3, 2)
# arr is:
# [[1 2]
#  [3 4]
#  [5 6]]

# Create a boolean mask
mask = arr > 3
# mask is:
# [[False False]
#  [False  True]
#  [ True  True]]

# Apply the mask
result = arr[mask]
# result is: [4 5 6]

Mastering indexing is key to effective data manipulation in NumPy.
